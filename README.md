# 🍔 Food Order App

A full-stack Food Ordering Web Application where users can browse food items, add them to cart, and place orders seamlessly.

---

## 🚀 Features

### 👤 User Features
- User Signup & Login (JWT Authentication)
- Browse food items
- Add to Cart
- Place Orders
- View Order History

### 🛠️ Admin Features (Coming Soon)
- Add / Edit / Delete Food Items
- Manage Orders
- Dashboard Analytics

---

## 🧰 Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Atlas)

### Authentication
- JWT (JSON Web Token)
- Bcrypt.js
